## 收录规则

### 来源
- 仅收录 wind8ai GitHub Stars 中标记为 `Harness` 列表的仓库
- 地址：https://github.com/stars/wind8ai/lists/harness
- **注意**：列表有分页，必须翻完所有页

### 收录条件
1. 以 wind8ai 的 star 为准，star 即收录，无需二次判断
2. 已公开（public repo）
3. Star ≥ 500（除非有特殊代表性）
4. 正式列表上限 **24 个**，按星数倒序；其余进候选池

### 排序
- 按 GitHub Star 总数倒序
- 星数仅展示量级（370k、182k...）

### 候选池
- 上限 **18 个**（正式 24 + 候选 18 = 42）
- 超出时按星数淘汰最低者

### 剔除规则
- 剔除某个项目时在对应条目**标题**旁注明 `❌ [YYYY-MM-DD] 原因`

### 不收录原则
- Star < 500 的项目（除非有特殊代表性）
