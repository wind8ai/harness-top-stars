## Inclusion Rules

### Source
- Only repos from wind8ai's GitHub Stars marked under the `Harness` list
- URL: https://github.com/stars/wind8ai/lists/harness
- **Note**: The list is paginated — all pages must be checked

### Inclusion Criteria
1. Based on wind8ai's stars — starred means included, no secondary judgment
2. Must be public repos
3. Star ≥ 500 (unless exceptionally representative)
4. Formal list capped at **24**, sorted by star count descending; rest go to candidates

### Sorting
- By total GitHub Stars descending
- Stars shown in magnitude format (375k, 211k...)

### Candidate Pool
- Capped at **24** (24 formal + 24 candidates = 48)
- When exceeded, lowest by stars are dropped

### Removal
- Removed projects are marked with `❌ [YYYY-MM-DD] reason` next to the entry title

### Exclusion
- Projects with < 500 stars (unless exceptionally representative)
