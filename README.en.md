## Inclusion Rules

### Source
- Only repos starred by wind8ai in the `Harness` list are included
- URL: https://github.com/stars/wind8ai/lists/harness
- **Note**: The list has pagination — all pages must be checked

### Criteria
1. Based on wind8ai's stars — starred means included
2. Public repository
3. Star ≥ 500 (unless exceptionally representative)
4. Main list capped at **24**, sorted by stars descending; rest go to candidates

### Sorting
- By total GitHub Stars, descending
- Stars shown as approximate magnitude (370k, 182k...)

### Candidates
- Capped at **18** (24 main + 18 candidates = 42)
- When full, lowest-star entries are dropped

### Removal
- Removed entries are annotated with `❌ [YYYY-MM-DD] reason`

### Exclusion
- Repos with < 500 stars (unless exceptionally representative)
